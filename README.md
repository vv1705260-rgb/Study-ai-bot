# StudySphere AI 🎓

**All-in-one study platform for Class 10 students**

Live Demo: https://yourname.github.io/studysphere-ai/

### ✨ Features
- **AI Chat Tutor** - 24/7 doubt solving with Hugging Face Llama-3
- **Community** - Post doubts, like, comment, follow classmates
- **Mock Tests** - 10-question timer tests with PDF certificates
- **Free Courses** - Google, Harvard, NPTEL, Microsoft with certificate links
- **Expert WhatsApp** - Direct chat with subject experts
- **Study Tools** - Pomodoro timer, flashcards, note saving, PDF export
- **Social Learning** - Reviews, follow, invite friends
- **Dark Mode** - Easy on eyes for night study

### 🚀 Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Hugging Face Inference API (Free AI)
- jsPDF for certificates
- LocalStorage for data persistence
- 100% Frontend - No backend needed

### 📱 How to Use
1. Open the live link
2. Sign up with any name/email
3. Explore tabs: Chat, Community, Mock Test, Courses
4. Replace `91**********` in code with your WhatsApp number
5. Add your name
